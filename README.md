# Financial Analysis App / 財務分析アプリ

## 概要

このアプリは、企業名・株式コード・ティッカーシンボルを入力すると、Yahoo Finance から財務データを取得し、主要な財務指標や財務三表をブラウザ上で確認できる Streamlit アプリです。

日本株・米国株の両方に対応しており、以下のような入力が可能です。

- `AAPL`
- `Apple`
- `MSFT`
- `Microsoft`
- `7203`
- `トヨタ`
- `SOFI`
- `ソーファイ`

日本株の4桁コードを入力した場合は、自動的に Yahoo Finance 用のティッカー形式に変換します。

例：

```text
7203 → 7203.T
6758 → 6758.T
